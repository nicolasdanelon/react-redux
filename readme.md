# useReducer demo

To run the php server you must:

```bash
php -S 127.0.0.1:9001
```

To run the project install pnpm:

```bash
npx pnpm i -g pnpm@latest
```

then just install the dependencies

```bash
pnpm install
```

and run the developement server

```bash
pnpm run dev
```
